Implement the full authentication and session management system for Ship's Express API. This covers session-cookie auth, API token auth, CSRF protection, and rate limiting.

Create `middleware/auth.ts` with an `authMiddleware` that: (1) checks for Bearer token first (API token auth via SHA-256 hash lookup in api_tokens table, validates not revoked/expired, updates last_used_at), (2) falls back to session cookie auth (reads `session_id` from cookies, validates session exists, checks 15-minute inactivity timeout and 12-hour absolute timeout per NIST SP 800-63B-4 AAL2, verifies workspace membership, updates last_activity, refreshes cookie with sliding expiration). Attach userId, workspaceId, isSuperAdmin, isApiToken to the Express Request. Also create `superAdminMiddleware`, `workspaceAdminMiddleware`, and `workspaceAccessMiddleware`.

Create `routes/auth.ts` with: POST /api/auth/login (email+password, bcrypt.compare, case-insensitive email lookup, creates cryptographically secure session via crypto.randomBytes(32), sets httpOnly/secure/sameSite=strict cookie, returns user+workspace data), POST /api/auth/logout (deletes session, clears cookie), GET /api/auth/me (returns current user+workspaces), POST /api/auth/extend-session, GET /api/auth/session. Create `routes/api-tokens.ts` for CLI token CRUD.

In `app.ts`, configure: helmet with CSP, CORS with credentials, csrf-sync (conditional: skip for Bearer tokens), express-rate-limit (login: 5 failed/15min with skipSuccessfulRequests, API: 100/min with Bearer exempt), session middleware, cookie-parser. The CSRF token endpoint GET /api/csrf-token must be before the CSRF middleware.

Use shared constants from a shared package: SESSION_TIMEOUT_MS=900000, ABSOLUTE_SESSION_TIMEOUT_MS=43200000, HTTP_STATUS, ERROR_CODES. All error responses must follow the shape `{ success: false, error: { code, message } }`. Log auth events to audit_logs via a `services/audit.ts` helper. Verify: run the API, login via curl, confirm cookie is set, confirm /api/auth/me returns user data, confirm expired sessions return 401 SESSION_EXPIRED.
